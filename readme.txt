idormWifiAtack.py: run nmap to scan for active MAC addresses and save them into a text file. Then this tool wil loop through these MAC attack and try spoofing them because some Wifi protocols authenticate users through MAC address.

run.sh: This tool comprises of a sequence of bash commands. In which they try to spoof DNS responses. To spoof DNS responses successfully, they must be unprotected with static and dynamic HSTS and the victims' browser/OS must not use DNS over Https.
